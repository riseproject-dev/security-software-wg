# RISE Security Software WG

## About RISE

[RISE](https://riseproject.dev) is a collaborative, industry-led initiative under the Linux Foundation that accelerates open-source software development for the RISC-V architecture.

## Charter

This working group focuses on tangible security-related work items for RISC-V - spanning firmware, TEEs, the Linux kernel, and toolchains - feeding into well-defined use cases such as enabling a Hardware Secure Root-of-Trust reference implementation and Confidential Compute support. Security is a horizontal function that spans multiple WGs. Our objectives are to:

- Help collaboration across companies to avoid duplication of work
- Accelerate security software development and upstreaming on RISC-V
- Communicate on the readiness of the respective ecosystems
